Github Cheatsheet

1. Maak een map:

Maak een map waar je alles in wilt hebben.


2. Git Status:

Check of er niet al een respository is gemaakt in je map door "git status" uit te voeren in cmd.


3. Git Init:

Maak een respository aan in je map door "git init" uit te voeren in cmd in je map.


4. Git Status:

Doe alweer "git status om te checken of de repository is gemaakt.


5. Maak een GitHub Repository:

Klik op het groene knopje waar "New" op staat.

Geef de repository een naam en klik dan op create om je repository aan te maken.


6. Linken aan GitHub:

Doe "git remote add origin (Repository link)" om je map te linken aan je GitHub repository.

Doe daarna "git branch -M main" om je branch op main te zetten.


7. Add & Commit:

Doe "git add ." om al je bestanden toe te voegen.

Doe daarna "git commit -m "commit name"" om alles te committen.


8. Pushen naar GitHub:

Doe "git push -u origin main" om alle bestanden te pushen naar je GitHub Repository.
